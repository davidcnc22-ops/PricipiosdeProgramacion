Ejercicio 10: Clasificación de un número
1. Declarar la variable numero
2. Solicitar al usuario el numero
3. Determinar si el numero es positivo si es mayor a o,
sino, determinar si es negativato si es menor a 0,
sino, determinar si es un 0
4. Imprimir el tipo de numero

# Declaracion de variables
numero = 0.0

# Entrada de datos
try:
    numero = float(input("Ingrese un numero: "))
    if numero > 0:
        print("El numero es positivo")
    elif numero < 0:
        print("El numero es negativo")
    else:
        print("El numero es cero")
except:
    print("Error: Ingrese un numero valido")
